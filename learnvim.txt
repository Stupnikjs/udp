gg debut du fichier
G fin ficher
dd delete
u undo
:w save 
:q quit
:number navigate to line
v select text

insert mode

Ctrl + D: Scrolls the screen down one line without moving the cursor.

Ctrl + F: Scrolls the screen down one page without moving the cursor

Ctrl + B: Scrolls the screen up one page without moving the cursor.

Ctrl + U: Delete from the cursor to begining of line.
